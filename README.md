# Enoch Mgijima Local Municipality Digital Platform

Modernized municipal portal for Komani (Queenstown), Eastern Cape, South Africa.

## Existing Structure (unchanged)
- `frontend/`
- `backend/`
- `database/`
- `admin-dashboard/`

## Frontend Upgrades
- Sticky municipal navigation with dropdowns and expanded government menu
- Hero carousel with municipal themed imagery and CTA actions
- Leadership cards, statistics counters, service grid, community news, and project tracker
- Citizen portal pages, issue reporting UX, notices/documents/tenders/quotations pages
- Smart map modules: `/map`, `/outages`, `/issue-map`, `/smart-city`
- Ward councillor directory and refuse collection schedule pages
- Council meetings livestream page and billing portal mock
- Floating municipal AI assistant widget

## Run
```bash
npm install
npm run dev:backend
npm run dev:frontend
```
